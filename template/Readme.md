```bash
npx create-elite-xpress my-app
```

```bash
cd my-app
```

```bash

```

## ✨ Features

- 🚀 Express.js with TypeScript
- 📁 Pre-configured folder structure
- 🛣️ Basic routing setup
- 🔒 CORS enabled
- ⚙️ Environment variables configuration
- 🔄 Hot reload development server
