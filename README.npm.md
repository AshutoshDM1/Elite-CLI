# Elite-xpress

## Description

Elite-xpress is simple boilerplate for building a rest api with express and typescript.

## Quick Start

```bash
npx create-elite-xpress my-app
```

```bash
cd my-app
```

```bash
npm run dev

# Visit `http://localhost:8000` to see your API in action!
```

## Build

```bash
npm run build
```

## Start

```bash
npm run start
```
