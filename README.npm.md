# Elite-xpress

## Description
Elite-xpress is simple boilerplate for building a rest api with express and typescript.

## Installation
```bash
npx create-elite-xpress my-app
```

## Usage

```bash
// cd to your project directory
cd my-app 

// install dependencies
npm install

// run the development server
npm run dev
```

## Build

```bash
npm run build
```

## Start

```bash
npm run start
```
